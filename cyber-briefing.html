Top Story

CISA keeps stacking the KEV list: a Magento RCE, a public Cisco SSRF exploit and a wave of actively exploited web flaws

Defenders are juggling a busy patch cycle as multiple critical bugs are exploited in the wild. CISA added CVE-2026-45247, a deserialization-to-RCE flaw (CVSS 9.8) in the Mirasvit Full Page Cache Warmer extension for Magento, to its Known Exploited Vulnerabilities catalog on June 3 after confirmed in-the-wild abuse; the fix landed in version 1.11.12. Separately, attackers are hammering CVE-2026-3300 (CVSS 9.8) in the Everest Forms Pro WordPress plugin to reach remote code execution and full site takeover.

On the enterprise side, Cisco confirmed that CVE-2026-20230, an SSRF-to-root bug in Unified Communications Manager (CVSS 8.6), is exploitable with publicly available exploit code, and warned that a high-severity Catalyst SD-WAN Manager flaw (CVE-2026-20245, CVSS 7.8) is under active attack. Google's June Android update, meanwhile, patched 124 issues including an exploited Framework integer-overflow bug (CVE-2025-48595, CVSS 8.4) now on the KEV list.
