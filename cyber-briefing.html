Active Exploitation · Domain Controllers

Windows Netlogon RCE (CVE-2026-41089) now exploited in the wild

A critical stack-based buffer overflow in Windows Netlogon — the service that handles authentication inside Windows domain environments — is under active exploitation, according to an updated advisory from the Centre for Cybersecurity Belgium (May 29). Microsoft's Security Response Center rates the flaw CVSS 9.8, with no sign-in or prior access required: a specially crafted network request to a server acting as a domain controller can lead to remote code execution.

The bug was patched on the May 12 Patch Tuesday alongside 136 other fixes. With domain controllers in the blast radius, unpatched Windows Server deployments should treat this as a priority-one update.

AI-built ransomware toolkit automates EDR evasion

A threat actor is using an AI-assisted ransomware toolkit that automates Active Directory discovery and helps evade EDR. Per BleepingComputer, development was assisted by coding agents across initial coding, analysis, and revision stages.

"Weedhack" MaaS targets Minecraft players

McAfee Labs codenamed a Minecraft-focused malware-as-a-service campaign "Weedhack" — active since January 2026, impersonating clients and mods, with 3,820 unique malicious JAR files and 240+ distribution URLs fueled by SEO poisoning and YouTube.
