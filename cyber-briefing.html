Top Story

Windows Netlogon RCE (CVE-2026-41089) under active exploitation — domain controllers at full-compromise risk

A critical stack-based buffer overflow in Windows Netlogon — the service that handles authentication inside Windows domains — is being actively exploited in the wild, with the Centre for Cybersecurity Belgium updating its advisory on May 29 to confirm in-the-wild attacks. Microsoft's Security Response Center rates it CVSS 9.8, with remote code execution and no user interaction required; unpatched domain controllers face full SYSTEM-level compromise with no credentials needed.

All versions from Windows Server 2012 through Windows Server 2025 are affected, including Server Core. Microsoft patched the flaw in its May 2026 Patch Tuesday cumulative updates — organizations that have not yet applied them should treat domain controllers as the top patching priority.

Sources: Help Net Security, SecurityWeek, Microsoft MSRC
Breaches & Incidents
Healthcare · Breach

DentaQuest breach exposes data of 2.6 million accounts

A data breach at the dental benefits administrator DentaQuest has reportedly exposed sensitive data tied to roughly 2.6 million accounts, one of the larger healthcare-sector incidents reported this month.

Humanitarian · Breach

UN World Food Programme app for Palestine breached

The WFP — the world's largest humanitarian organization — revealed that its self-registration application (SRA) for Palestine was breached, raising concerns about exposure of data belonging to vulnerable aid recipients.

Supply Chain · Malware

"IronWorm" infostealer infects 36 npm packages

A new supply-chain attack has seeded 36 packages on the npm registry with an infostealer dubbed IronWorm, the latest in a string of attacks targeting open-source package ecosystems used by developers.

Threat Actor · Cloud

"PCPJack" turns hijacked cloud servers into covert spam relay

The threat actor known as PCPJack has hijacked cloud servers on AWS, Google Cloud, and Azure to build a covert SMTP email relay network, quietly converting compromised business servers across the U.S., Europe, and Asia into spam proxies.
