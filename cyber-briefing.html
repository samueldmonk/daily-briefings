Top Story

ShinyHunters leaks claimed 234 GB from DentaQuest as breach exposes data on 2.6 million accounts

Dental benefits administrator DentaQuest — which serves 35 million customers across programs in 50 states with a network of 140,000 dentists — confirmed on June 2 that its networks had been breached. The extortion group ShinyHunters, which listed the company on its leak site last month claiming more than 234 GB of stolen data, publicly leaked the trove after what it described as a failure to reach an agreement with the company.

The exposed data on roughly 2.6 million accounts includes names, addresses, email addresses, phone numbers, dates of birth, government-issued IDs and health insurance information. Much of it appeared in healthcare enrollment files (ASC X12 transaction sets), some containing Medicaid IDs. DentaQuest said the incident caused "limited disruption" to customer service.

UN World Food Programme app breached

SUPPLY CHAIN

"IronWorm" infostealer infects 36 npm packages

A new supply-chain attack seeded 36 packages on the npm registry with infostealer malware dubbed IronWorm — the latest in a steady drumbeat of open-source registry compromises targeting developers.

CLOUD

"PCPJack" turns hijacked cloud servers into spam relays

A threat actor tracked as PCPJack hijacked cloud servers associated with AWS, Google Cloud and Microsoft Azure to build a covert SMTP email relay network, converting compromised business servers into SMTP proxies.

RANSOMWARE

AI-built ransomware toolkit automates EDR evasion

On June 2, researchers detected a threat actor using an AI-assisted ransomware toolkit that automates Active Directory discovery and helps evade EDR. Development was reportedly assisted by AI coding agents across initial coding, analysis and revision stages.

DEV TOOLS

One-click VS Code attack can steal GitHub tokens

Researchers disclosed a one-click attack via Microsoft Visual Studio Code that lets an attacker steal a user's GitHub token — "just by clicking a link," per the researcher who reported it.
