Top Story

Actively exploited critical flaws stack up: Magento cache, WordPress forms and an unpatched Exchange spoofing bug

Defenders face a busy week as multiple critical vulnerabilities are being exploited in the wild. CISA added CVE-2026-45247, a deserialization flaw (CVSS 9.8) in the Mirasvit Full Page Cache Warmer extension for Magento, to its Known Exploited Vulnerabilities catalog following confirmed in-the-wild abuse. Separately, attackers are hammering CVE-2026-3300 (CVSS 9.8) in the Everest Forms Pro WordPress plugin to achieve remote code execution and full site takeover.

Microsoft, meanwhile, disclosed exploitation of CVE-2026-42897, an Exchange Server spoofing/XSS issue in Outlook Web Access rated 8.1, for which no patch yet exists — the Exchange Emergency Mitigation Service applies automatic mitigation by default. Cisco also warned that a high-severity Catalyst SD-WAN Manager bug is under active attack with no fix available.
