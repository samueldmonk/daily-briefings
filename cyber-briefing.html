Top Story

Windows Netlogon RCE (CVE-2026-41089) under active exploitation — unpatched domain controllers face full takeover

Belgium's Centre for Cybersecurity (CCB) warned on May 29 that CVE-2026-41089, a critical pre-authentication stack-based buffer overflow in the Windows Netlogon service (CVSS 9.8), is being actively exploited in the wild against unpatched Windows domain controllers. An unauthenticated attacker can send a crafted network request to a domain controller and gain SYSTEM-level code execution — in practice, full control of the Active Directory domain, the ability to create privileged accounts, and lateral movement across every system that authenticates against that controller.

Microsoft patched the flaw in its May 12 Patch Tuesday updates. Organizations that have not yet deployed the May cumulative update for Windows Server should treat it as an emergency change.

Breaches & Incidents
BREACHHEALTHCARE

DentaQuest: 2.6M accounts exposed

The Sun Life–owned dental benefits administrator confirmed June 2 that its network was breached after extortion group ShinyHunters leaked a claimed 234 GB of data. Exposed records include 2.6 million unique email addresses plus names, mailing addresses, dates of birth, phone numbers and health insurance information; Social Security numbers do not appear to have been taken.

BREACHHUMANITARIAN

UN World Food Programme app breached

The WFP, the world's largest humanitarian organization, revealed that its self-registration application (SRA) for Palestine was breached, exposing data tied to aid registration.

CLOUDHIJACK

"PCPJack" turns hijacked cloud servers into spam relay

A threat actor hijacked cloud servers associated with AWS, Google Cloud, and Microsoft Azure to build a covert SMTP email relay network — compromised business servers across the U.S., Europe, and Asia were quietly converted into SMTP proxies.

SUPPLY CHAIN

36 npm packages infected with "IronWorm" infostealer

A new supply-chain attack seeded 36 packages on the npm registry with infostealer malware dubbed IronWorm. Separately, researchers disclosed a one-click VS Code attack that can steal a user's GitHub token — with read/write access to private repos — just by clicking a link.
