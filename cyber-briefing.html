Top Story

Windows Netlogon RCE (CVE-2026-41089) under active exploitation — domain controllers at risk

A critical stack-based buffer overflow in Windows Netlogon — the service that handles authentication inside Windows domains — is now being actively exploited in the wild. CVE-2026-41089 (CVSS 9.8) lets an unauthenticated attacker send a crafted network request to a domain controller and execute code without any sign-in or prior access. Microsoft patched the flaw on May 12 as part of its Patch Tuesday updates, crediting its internal WARP research team.

The Centre for Cybersecurity Belgium updated its advisory on May 29 to confirm in-the-wild exploitation — notable because Microsoft had originally rated the bug "exploitation less likely." Successful exploitation yields SYSTEM-level code execution on the domain controller itself: in practice, full control of the Active Directory domain, privileged account creation, and lateral movement to everything that authenticates against it. If your DCs aren't patched, treat this as an emergency.

Breaches & Incidents
DATA BREACH

DentaQuest: 2.6M accounts exposed

A breach at dental benefits administrator DentaQuest has reportedly exposed sensitive data tied to roughly 2.6 million accounts, making it one of the larger healthcare-sector incidents disclosed this period.

HUMANITARIAN

UN World Food Programme app breached

The UN's World Food Programme — the world's largest humanitarian organization — revealed that its self-registration application (SRA) for Palestine was breached, raising concerns about exposed data of vulnerable beneficiaries.

SUPPLY CHAIN

npm "IronWorm" infostealer hits 36 packages

A new supply-chain attack infected 36 packages on the npm registry with infostealer malware dubbed IronWorm. Separately, researchers disclosed a one-click VS Code attack capable of stealing a user's GitHub token.

RANSOMWARE

Abyss hits German county; AI-built tooling spreads

Germany's Limburg-Weilburg county administration was hit by the Abyss ransomware group, which claims ~132 GB stolen and encrypted. Meanwhile BleepingComputer reports a threat actor using an AI-built ransomware toolkit that automates Active Directory discovery and EDR evasion.
