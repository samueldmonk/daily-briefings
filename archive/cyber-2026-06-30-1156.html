Top Story

Klue OAuth Breach Exposes 100+ Salesforce Instances to Icarus Ransomware

Business intelligence platform Klue suffered a supply-chain attack that gave the Icarus threat group access to customer OAuth tokens, enabling wholesale Salesforce data theft. Attackers pushed a malicious code update to Klue's backend on or around June 11, stealing tokens for third-party integrations; by June 19, Icarus publicly claimed the breach and began exfiltrating CRM data on June 23. At least 12 organizations confirmed compromised: Gong, HackerOne, Huntress, Insurity, Jamf, LastPass, OneTrust, Recorded Future, Snyk, Sprout Social, and Tanium. A second, unnamed group has since stolen the same data from Icarus.

⚡ Patch Priority

CVE-2026-43503 (DirtyClone): Linux kernel local privilege escalation (CVSS 8.8) allowing any unprivileged user to gain root. Public proof-of-concept released June 25. Patch immediately on multi-tenant systems, Kubernetes nodes, and any host with unprivileged namespaces. Linux kernel v7.1-rc5+ has the fix (landed May 21).

CVE-2026-55200 (libssh2): Client-side RCE (CVSS 9.2) in all versions ≤1.11.1, affects curl, Git, PHP. A malicious SSH server can corrupt client memory with no credentials required. Public PoC available. Patch immediately on developer workstations, CI/CD pipelines, and all affected tools.

Breaches & Incidents

Multiple Data Breaches Disclosed June 30

Five organizations disclosed breaches on June 30, 2026: Abans Financial Services, Agroprime, Bonacio Inc., Boston Orthotics & Prosthetics, and Bristol Place Corporation. Details remain limited; investigation ongoing.

NEW

ServiceNow Unauthenticated API Exploit Disclosed

An unauthenticated REST endpoint (/api/now/related_list_edit/create) in ServiceNow was exploited starting ~June 2–3. The endpoint had requires_authentication=false set in configuration. Instances were patched June 5; primarily affected Australia platform release and older configurations. Attack indicator: IP 51.159.98.241.

Hijacked npm, Go Packages Deliver Information Stealer

Security researchers uncovered two hijacked npm packages and a cluster of malicious Go modules designed to drop a Python-based information stealer on Windows, Linux, and macOS. Supply-chain attack; likely development environment targeting.
