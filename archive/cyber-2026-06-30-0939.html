Patch Priority: Windows Kernel RCE

CVE-2026-45657 — Windows Kernel remote code execution (CVSS 9.8) allowing unauthenticated remote attackers to execute code at SYSTEM level without user interaction. Patch immediately: included in Microsoft June 2026 security update.

Top Story: Agentjacking Emerges as AI Supply-Chain Risk

Tenet Security and the Cloud Security Alliance disclosed a novel attack class called "agentjacking" in which malicious Sentry error events are crafted to appear as trusted output to AI coding agents (Claude Code, Cursor, etc.) via the Sentry MCP server, leading to direct code execution. Sentry declined a root-level fix, calling it "technically not defensible" at the platform level. The attack bypasses traditional EDR, WAF, and IAM controls. This represents an emerging supply-chain risk class targeting AI-assisted development workflows.

Threat Actor Spotlight: Icarus OAuth Pivot

The Icarus ransomware group executed a novel OAuth attack against Salesforce integrator Klue, extracting CRM data from downstream clients including HackerOne, Gong, OneTrust, Tanium, and Huntress. The technique demonstrates how identity and access mechanisms can be weaponized in the SaaS supply chain when a single integration point is compromised.
