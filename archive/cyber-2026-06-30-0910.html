Top Story

ShinyHunters Exploits Oracle PeopleSoft Zero-Day Across Education Sector
The ShinyHunters cybercrime group has exploited CVE-2026-35273 (CVSS 9.8), a critical remote code execution vulnerability in Oracle PeopleSoft Enterprise PeopleTools, to compromise servers at 100+ organizations, with universities and higher education institutions bearing the brunt. The zero-day required no authentication or user interaction; attackers staged customized MeshCentral agents to execute commands and deploy lateral movement scripts. Stolen data published on ShinyHunters' leak site includes 454,600+ records from the University of Nottingham alone.
Patch Priority
Microsoft Office RCEs (CVE-2026-45461, 45463, 45472, 45474) — CVSS 8.4 use-after-free and integer-underflow flaws in Preview Pane allowing arbitrary code execution via malicious documents. June 2026 Patch Tuesday. Currently exploited in the wild via malicious Office files.

Breaches & Incidents

Temu Breach: 310M User Records Posted for Sale New
An alleged 310 million Temu user records have been listed for sale on a cybercrime forum; researchers note the scale remains unverified. If legitimate, would constitute one of the largest data breaches of 2026.
Klue Supply Chain Attack via Salesforce New
Market intelligence platform Klue disclosed that hackers breached its Salesforce integration on June 11–12, stealing data from an undisclosed number of customer Salesforce environments in a supply chain attack.
libssh2 Remote Code Execution (CVE-2026-55200)
Critical flaw allows malicious SSH servers to trigger memory corruption and code execution on connecting clients (CVSS 9.2). Affects every release up to 1.11.1; embedded in curl, Git, PHP, backup agents, firmware updaters, and appliances worldwide.
DirtyFrag Variant (CVE-2026-43503) Gains Root on Linux
New variant of DirtyFrag allows unprivileged local users to manipulate the Linux page cache and escalate privileges to root; affects Linux systems lacking current patches.

Threat Actor Spotlight

ShinyHunters: From Marketplace Breaches to Zero-Day Exploitation
ShinyHunters has evolved from opportunistic marketplace and credential-stealing campaigns to sophisticated zero-day exploitation targeting critical enterprise infrastructure. The group's shift toward Oracle PeopleSoft and other ERP systems reflects higher operational maturity and a focus on high-value victims, particularly in the education sector where data inventory (student records, financial info) is extensive.
