Top Story

Progress tells ShareFile customers: shut down your Storage Zone Controllers

Progress Software has told ShareFile customers to shut down the Windows servers running their Storage Zone Controllers, confirming to The Hacker News that it is responding to a "credible external security threat." The company temporarily disabled access to affected accounts "out of an abundance of caution" while internal and external security experts investigate, and says it has no indication of unauthorized access to any ShareFile accounts or data.

What Progress has not said is what the threat is or who is behind it. The order became public when a customer posted the company's email to Reddit's r/sysadmin on July 10; Progress confirmed the disruption on its status page, listing Storage Zone Controller customers as "not operational" with the incident under investigation. Cloud services have since been gradually restored, but Progress is telling customers to keep Storage Zone Controllers offline (SecurityWeek/The Register). Only the Storage Zone Controller component is affected — but for file-transfer infrastructure, an emergency power-down order is the kind of signal defenders shouldn't wait to act on.

Patch Priority

Due TODAY: CVE-2026-48939 (Joomla iCagenda) and CVE-2026-56291 (Balbooa Joomla Forms) — both CVSS 10.0, both exploited as zero-days

CISA's risk-based remediation date for both Joomla-extension flaws is today, July 13 (0 days left). Both allow arbitrary file upload leading to remote code execution; iCagenda exploitation has been underway since June 15 in automated attacks (mySites.guru via The Hacker News). Verified fixes: iCagenda 4.0.8 (or 3.9.15 for the legacy branch) and Balbooa Forms 2.4.1. If you run Joomla with either extension and can't patch now, take the site offline — exploitation is active and trivial.

Threat Actor Spotlight

Forg365 — phishing-as-a-service with an enterprise feature list

A new PhaaS operation dubbed Forg365 is targeting Microsoft 365 accounts with a blend of device-code phishing, adversary-in-the-middle session theft, antibot evasion, AI-assisted lure creation and post-compromise mailbox operations. Sold via Telegram for $400 a month (or $3,800 a year), its lures ride legitimate email infrastructure such as Amazon SES and Twilio SendGrid before landing on operator-controlled domains. Researchers at ZeroBAC describe a mature operator panel — OAuth app configuration, SMTP rotation, AI email generation, token vaulting and keyword alerts — a reminder that commodity phishing now ships with product-grade tooling (The Hacker News, Jul 13).

Breaches & Incidents

NewAI tradecraft

Attacker maps Active Directory with suspected AI-written script

Huntress flagged an intrusion in which a threat actor used a "vibe-coded" PowerShell script to enumerate Active Directory — finding the domain controller, mapping users, computers and domains, then generating an AD_Report.html to measure its own success. The attacker RDP'd into a domain-joined Windows Server with pre-compromised credentials and staged tools in C:\ProgramData\ in the early-June incident; telltale AI signs included a prompt-iteration title, placeholder strings and over-engineered code (Huntress' Jevon Ang and Dray Agha via The Hacker News, Jul 13).

AI security

MemGhost: one email plants false memories in AI agents

Researchers demonstrated a "stealth memory injection" attack in which a single crafted email tricks a memory-enabled AI assistant into saving a false "fact" about its user, hiding the change, and steering later sessions — with a tool that writes the attack emails automatically. The paper landed on arXiv July 6 (The Hacker News, Jul 13).

supply chain

jscrambler npm release drops infostealer on install

The compromised jscrambler 8.14.0 npm release, published July 11, carries a preinstall hook that drops a native infostealer binary for Windows, macOS or Linux. Socket flagged it six minutes after publication; anything that pulled it in that window ran the payload with the install process's access. Version 8.13.0 is clean (The Hacker News).

phishing ops

Misconfigured server exposes three Evilginx phishing operations

A phishing operator left a Python web server with directory listing open on a public port; French firm Lexfo lifted the entire toolkit and pivoted to two more operators — three Microsoft 365 campaigns in all, each running a custom Evilginx fork, the largest active for over a year against corporate mailboxes (The Hacker News, Jul 13).

espionage

Pakistani police portals hit in multi-group espionage

SentinelOne detailed sustained espionage against Pakistani law-enforcement bodies by suspected China- and India-aligned actors from February 2024 to April 2026 — including compromise of Balochistan Police servers holding criminal, biometric and personnel records, with a custom implant disguised as a portal update (The Hacker News).

breach

Accenture breach claims: "888" hawks 35GB haul

Accenture has acknowledged a security incident after threat actor "888" offered roughly 35GB of claimed loot — source code, with SSH keys and Azure tokens also claimed by the seller — while disputing the scale and calling it an isolated, remediated matter. The same handle previously touted Accenture employee data in 2024 (BleepingComputer and others; key/token claims are the seller's, unconfirmed).
