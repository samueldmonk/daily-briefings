TOP STORY

Agentjacking: AI Coding Agents Under Attack

Security researchers at Tenet Security disclosed a new supply-chain vulnerability dubbed "agentjacking" that targets AI coding assistants like Claude Code and Cursor. Attackers craft malicious error events that are fed to the agents via the Sentry MCP server, which the agents treat as trusted output. This allows arbitrary code execution without triggering EDR, WAF, or IAM defenses—a novel attack class on the AI supply chain. Sentry declined a platform-level fix and characterized the issue as "technically not defensible" at their layer. Disclosed Jun 3; no public PoC or mass exploitation reported to date.

THREAT ACTOR SPOTLIGHT

ShinyHunters — Multi-Vector Extortion

ShinyHunters have expanded from credential theft to large-scale ransomware operations, targeting high-profile organizations (MSG Sports, recent reports). The group leverages OAuth exploitation to gain initial access, escalates to lateral movement, and exfiltrates massive PII repositories before deploying ransomware. Defenders should prioritize OAuth/federated auth hardening, privileged access management (PAM), and cloud API audit logging.
